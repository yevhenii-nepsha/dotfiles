# Moonfly Theme for Telegram iOS

Manual setup guide for the Moonfly color scheme on Telegram for iPhone/iPad.

## Installation

1. Open Telegram → **Settings** → **Appearance**
2. Scroll down and tap **Create New Theme**
3. Configure each tab as described below

## Color Configuration

### Background Tab
| Setting | Color |
|---------|-------|
| Background | `#1c1c1c` |

### Accent Tab
| Setting | Color |
|---------|-------|
| Accent | `#80a0ff` |

### Messages Tab

For a consistent dark look, set all color slots to:

| Slot | Color |
|------|-------|
| Color 1 | `#1e2530` |
| Color 2 | `#1e2530` |
| Color 3 | `#1e2530` |
| Color 4 | `#1e2530` |

> **Tip:** iOS automatically adjusts incoming message bubbles to be slightly darker than the background.

## Moonfly Palette Reference

| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#1c1c1c` | Main background |
| Surface | `#1c1c1c` | Elevated surfaces |
| Blue | `#80a0ff` | Primary accent |
| Bright Blue | `#78c2ff` | Hover states |
| Green | `#8cc85f` | Success |
| Emerald | `#42cf89` | Online status |
| Red | `#ff5454` | Errors, attention |
| Coral | `#e2637f` | Secondary accent |
| Yellow | `#e3c78a` | Warnings |
| Cyan | `#7ee0ce` | Code blocks |
| Violet | `#ae81ff` | Special elements |
| Text | `#b2b2b2` | Primary text |
| Muted | `#9e9e9e` | Secondary text |
